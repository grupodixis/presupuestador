# Checklist de cualificación de proveedores

## Identificación

- [ ] Nombre comercial identificado.
- [ ] Persona de contacto identificada.
- [ ] Teléfono disponible.
- [ ] Email disponible.
- [ ] Web o catálogo disponible.
- [ ] Ubicación y zona de servicio conocidas.

## Capacidad técnica

- [ ] Suministra el material o servicio requerido.
- [ ] Puede trabajar con las medidas o formatos necesarios.
- [ ] Puede aportar certificados si son necesarios.
- [ ] Puede ofrecer alternativas equivalentes.
- [ ] Tiene experiencia en producto similar.

## Condiciones económicas

- [ ] Precio unitario claro.
- [ ] Precio total claro.
- [ ] IVA indicado.
- [ ] Transporte indicado.
- [ ] Descuentos por volumen indicados si aplica.
- [ ] Validez de oferta indicada.

## Plazo y logística

- [ ] Disponibilidad actual indicada.
- [ ] Plazo de entrega indicado.
- [ ] Lugar de entrega confirmado.
- [ ] Condiciones de descarga indicadas si aplica.

## Fiabilidad

- [ ] Responde en plazo razonable.
- [ ] La respuesta es clara y completa.
- [ ] Tiene referencias o trabajos verificables.
- [ ] No exige condiciones incompatibles con el proyecto.
- [ ] Se puede volver a contactar en futuros presupuestos.

## Decisión

- [ ] Validar como proveedor útil.
- [ ] Mantener como candidato pendiente.
- [ ] Descartar indicando motivo.

## Motivos habituales de descarte

- No suministra en Menorca.
- Precio fuera de mercado.
- Plazo incompatible.
- No responde.
- No puede garantizar especificación técnica.
- Transporte demasiado costoso.
