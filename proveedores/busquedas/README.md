# Búsquedas de proveedores

Registro de búsquedas realizadas para encontrar nuevos proveedores.

## Cuándo crear un registro

- Cuando no existe proveedor validado para una partida.
- Cuando el proveedor habitual no responde.
- Cuando el precio recibido parece alto y se necesitan alternativas.
- Cuando aparece un material nuevo no registrado.

## Formato recomendado

Crear un archivo por búsqueda con nombre:

`YYYY-MM-DD-categoria-material.md`

Ejemplo:

`2026-05-22-madera-peldanos-roble.md`

## Contenido mínimo

- Necesidad detectada.
- Palabras clave usadas.
- Proveedores encontrados.
- Datos de contacto.
- Estado de contacto.
- Decisión: crear ficha / descartar / pendiente.
