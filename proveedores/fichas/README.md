# Fichas de proveedores

Crear aquí una ficha por proveedor usando `../plantilla-ficha-proveedor.md`.

## Convención de nombres

Usar nombres en minúscula, sin espacios y con categoría si ayuda:

- `vidrio-proveedor-ejemplo.md`
- `madera-proveedor-ejemplo.md`
- `metal-proveedor-ejemplo.md`
- `tratamiento-proveedor-ejemplo.md`

## Información sensible

No registrar datos personales innecesarios. Mantener solo datos de contacto profesional necesarios para solicitar presupuestos.
