### Regla practica para accesorios

- Para corredera E150 con herraje completo alto, estimar accesorios como **100-105% del coste de perfiles**.
- Para variante mas sencilla, estimar accesorios como **70% del coste de perfiles**.
- Si no se conoce el herraje, usar **85% del coste de perfiles** como valor intermedio prudente.

## Reglas de uso por el agente

1. Si se presupuestan perfiles EXTRUAL lacados y no hay precio actualizado, usar **11-12 EUR/kg comprado** como referencia rapida de compra.
2. Para una unidad de carpinteria terminada calculada por peso neto, usar **16-17 EUR/kg neto** antes de vidrio, mano de obra y margen.
3. Si el color no es blanco estandar, sumar suplemento o pedir precio real al proveedor.
4. Para ambiente exterior en Menorca, confirmar que el lacado es apto para ambiente marino y que tornilleria/accesorios expuestos son adecuados.
5. Actualizar este historico con cada presupuesto real de Perfila Menorca, Maderesa, Cortizo u otros proveedores de aluminio.
