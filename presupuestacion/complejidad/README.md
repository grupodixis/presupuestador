# Complejidad

Evaluación de la complejidad de fabricación y montaje para ajustar costes y márgenes.

## Factores de complejidad

### Complejidad geométrica

| Factor | Peso | Aplica a |
|---|---|---|
| Forma irregular o no rectilínea | Alto | barandillas, escaleras |
| Múltiples ángulos y cortes | Alto | estructuras, cerramientos |
| Superficies curvas | Muy alto | cualquier producto |
| Tolerancias estrechas | Alto | carpintería, cerramientos |
| Mediciones en obra sin planos | Medio | cualquier montaje |

### Complejidad de materiales

| Factor | Peso |
|---|---|
| Material difícil de mecanizar (inoxidable) | Alto |
| Material con protección especial | Medio |
| Material con certificación de origen | Medio |
| Varios materiales combinados | Alto |

### Complejidad de montaje

| Factor | Peso |
|---|---|
| Altura superior a 6 metros | Alto |
| Acceso difícil | Alto |
| Montaje con ocupación (cliente presente) | Medio |
| Necesidad de andamios/plataformas | Medio |
| Varios frentes de trabajo | Medio |
| Plazo de montaje ajustado | Alto |

### Complejidad de procesos

| Factor | Peso |
|---|---|
| Soldadura especial (aluminio, inox) | Alto |
| Tratamiento superficial multiple | Medio |
| Mecanizado de precisión | Alto |
| Control de calidad externo | Medio |

## Cálculo del factor de complejidad

1. Identificar los factores aplicables.
2. Sumar pesos: Bajo=1, Medio=2, Alto=3.
3. Obtener puntuación total.
4. Aplicar factor de ajuste al coste:

| Puntuación | Factor de ajuste |
|---|---|
| 0-3 | 1.0 (estándar) |
| 4-6 | 1.1 - 1.2 |
| 7-10 | 1.2 - 1.4 |
| 11-15 | 1.4 - 1.7 |
| >15 | 1.7 - 2.0 |

## Uso

El agente debe evaluar la complejidad después de identificar los procesos necesarios y antes de aplicar márgenes. El factor de complejidad multiplica el coste técnico base.
