# Oferta y Demanda

Ajustes de precio basados en la capacidad productiva actual y las condiciones del mercado.

## Indicadores de capacidad

| Indicador | Ajuste |
|---|---|
| Taller por debajo del 60% de carga | Reducir margen 5-10% para captar trabajo |
| Taller al 60-80% de carga | Margen estándar |
| Taller al 80-100% de carga | Aumentar margen 5-10% |
| Taller saturado (>100%) | Aumentar margen 15-20% o rechazar |

## Indicadores de demanda

| Situación | Ajuste |
|---|---|
| Cliente con urgencia | Recargo 10-20% |
| Cliente sin presión de plazo | Margen estándar |
| Repetido del mismo cliente | Posible descuento 5% |
| Obra nueva gran volumen | Descuento 5-10% |
| Reforma pequeña | Margen estándar o superior |
| Temporada baja (agosto, navidad) | Reducir margen para mantener carga |
| Temporada alta | Aumentar margen |

## Reglas de decisión

1. Si la carga de trabajo es baja, priorizar captación sobre margen.
2. Si la carga es alta, priorizar rentabilidad sobre volumen.
3. Si el cliente es estratégico, mantener precio competitivo.
4. Si el trabajo requiere certificación externa, considerar recargo.
5. Si hay riesgo de penalización por plazo, aumentar margen.

## Referencia temporal

Los ajustes por estacionalidad deben revisarse mensualmente. Se puede crear un archivo `estacionalidad.json` con los factores históricos mes a mes.
