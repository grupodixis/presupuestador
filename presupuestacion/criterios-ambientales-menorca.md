# Criterios Ambientales - Menorca

## Regla base

Todos los presupuestos se consideran por defecto para **Menorca**, salvo que el usuario indique expresamente otra ubicacion.

Menorca debe tratarse como un entorno de alta corrosion por:

- Isla con exposicion continua a brisa marina.
- Ambiente salino y oxidante.
- Alta humedad ambiental.
- Cercania frecuente a costa, piscinas, jardines con riego y zonas exteriores.

## Clasificacion practica

| Situacion | Criterio de ambiente |
|---|---|
| Interior seco y protegido | C3 minimo por contexto insular, salvo vivienda muy controlada. |
| Exterior urbano/interior de isla | C4 recomendado. |
| Exterior expuesto, costa, terraza, balcon, piscina o cubierta | C5-M / ambiente marino severo. |
| Contacto directo con agua salina, primera linea o salpicaduras | C5-M severo; evitar soluciones debiles. |

## Materiales recomendados por defecto

| Material / sistema | Uso recomendado |
|---|---|
| Acero inoxidable AISI 316 | Preferente en exteriores, costa, barandillas vistas, tornilleria y fijaciones. |
| Acero inoxidable AISI 304 | Solo interior o exterior muy protegido. No usar por defecto en ambiente marino expuesto. |
| Acero al carbono galvanizado en caliente | Opcion robusta para exterior si el diseno permite galvanizar correctamente. |
| Acero galvanizado + sistema de pintura adecuado | Recomendado cuando se requiera color RAL en exterior. |
| Aluminio lacado marino / calidad exterior | Valido para carpinteria, cerramientos y piezas ligeras si el sistema es apto para ambiente marino. |
| Acero pintado sin galvanizar | Evitar en exterior de Menorca salvo trabajos temporales o bajo aceptacion expresa del cliente. |

## Tornilleria y fijaciones

- En exterior, usar tornilleria inoxidable A2 como minimo y A4 preferentemente en costa o ambiente muy salino.
- No mezclar metales sin considerar corrosion galvanica.
- En anclajes a obra exterior, valorar taco quimico, varilla inox y sellado correcto.

## Tratamientos superficiales

- Para acero al carbono exterior, presupuestar como minimo galvanizado en caliente o sistema epoxi + poliuretano adecuado a C4/C5.
- Para ambiente marino severo, no presupuestar imprimacion simple + esmalte como solucion definitiva.
- En piezas soldadas, incluir repaso, limpieza y proteccion de cordones y zonas afectadas por calor.
- Si se pinta sobre galvanizado, incluir preparacion compatible para adherencia.

## Regla para el agente

Antes de calcular un presupuesto, el agente debe preguntar o asumir:

1. Si la pieza va en interior o exterior.
2. Si esta cerca de costa, piscina, riego o humedad permanente.
3. Si el cliente exige color RAL o acepta acabado inoxidable/galvanizado.
4. Si hay tornilleria o anclajes expuestos.
5. Si el material propuesto resiste el ambiente de Menorca.

Si el usuario no especifica ubicacion, usar por defecto: **Menorca, ambiente salino, criterio C4/C5 segun exposicion**.

## Advertencia comercial

Cuando se descarte una solucion economica por corrosion, explicarlo de forma simple al cliente:

> Por tratarse de una instalacion en Menorca, se ha previsto una solucion apta para ambiente marino y alta corrosion, evitando acabados que puedan oxidarse prematuramente.
