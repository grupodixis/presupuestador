# Márgenes

Política de márgenes industriales y comerciales según tipo de producto, cliente y volumen.

## Factores que afectan al margen

| Factor | Afecta |
|---|---|
| Tipo de producto | Complejidad ↦ margen mayor |
| Volumen | Series largas ↦ margen menor |
| Cliente habitual | Descuento por fidelidad |
| Cliente nuevo | Margen estándar o promocional |
| Urgencia | Recargo por plazo reducido |
| Riesgo técnico | Prima por incertidumbre |
| Competencia | Ajuste para ser adjudicado |

## Reglas de ajuste

- Sumar 5-10% si el plazo de entrega es inferior al estándar.
- Restar 5% si es cliente habitual con volumen anual > 50.000 €.
- Sumar 5-15% si hay riesgo técnico significativo.
- Sumar 5-10% si se requiere certificación o garantía especial.
- Restar 5-10% si es una licitación pública con competencia alta.
