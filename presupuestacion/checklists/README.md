# Checklists

Listas de verificación para asegurar que el presupuesto es completo, correcto y defendible.

## Checklist general de presupuesto

- [ ] Tipo de producto identificado correctamente.
- [ ] Skill correspondiente cargada y aplicada.
- [ ] Composición del producto consultada.
- [ ] Materiales principales listados y cuantificados.
- [ ] Materiales auxiliares incluidos.
- [ ] Mano de obra de taller estimada.
- [ ] Mano de obra de montaje estimada.
- [ ] Maquinaria necesaria identificada.
- [ ] Tratamientos superficiales incluidos.
- [ ] Transporte contemplado.
- [ ] Mermas aplicadas.
- [ ] Complejidad evaluada y factor aplicado.
- [ ] Riesgos evaluados y factor aplicado.
- [ ] Margen aplicado según política.
- [ ] Ajuste de oferta/demanda aplicado.
- [ ] Impuestos incluidos/indicados.
- [ ] Plazo de entrega indicado.
- [ ] Condiciones de pago indicadas.
- [ ] Validez del presupuesto indicada.

## Checklist técnica

- [ ] Normativa aplicable verificada.
- [ ] Tolerancias de fabricación consideradas.
- [ ] Ambiente de exposición considerado (interior/exterior/C1-C5).
- [ ] Sobrecargas y resistencias verificadas.
- [ ] Tratamiento anticorrosión adecuado al ambiente.
- [ ] Anclajes y fijaciones dimensionados.
- [ ] Juntas y sellados considerados.
- [ ] Dilatación térmica considerada (aluminio, grandes luces).

## Checklist comercial

- [ ] Tipo de cliente identificado (particular, empresa, administración).
- [ ] Historial del cliente consultado si existe.
- [ ] Cliente estratégico o no.
- [ ] Competencia identificada.
- [ ] Urgencia del cliente considerada.
- [ ] Capacidad actual del taller considerada.
- [ ] Presupuesto dentro del rango de mercado estimado.

## Checklist de presentación

- [ ] Formato claro y profesional.
- [ ] Partidas desglosadas y explicadas.
- [ ] Mediciones visibles.
- [ ] Datos de contacto incluidos.
- [ ] Número de presupuesto y fecha.
- [ ] Firma/responsable indicado.
- [ ] Notas aclaratorias si procede.

## Uso

El agente debe ejecutar la checklist general antes de dar por finalizado cualquier presupuesto. Las checklists técnica y comercial se aplican según el producto y cliente.
