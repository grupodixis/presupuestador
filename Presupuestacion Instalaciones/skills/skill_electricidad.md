# Skill: Conteo de Electricidad

## Cuándo usar esta skill

Para contar instalaciones electricas de baja tension: alumbrado, fuerza, mecanismos, cuadros, circuitos, canalizaciones y tomas especiales.

## Unidades habituales

| Elemento | Unidad | Criterio de conteo |
|---|---:|---|
| Punto de luz | punto | Cada salida a luminaria o grupo controlado |
| Luminaria | ud | Cada equipo de iluminacion suministrado o instalado |
| Interruptor / conmutador / pulsador | ud | Cada mecanismo de mando visible |
| Enchufe monofasico | ud | Cada toma utilizable, aunque comparta marco |
| Toma trifasica | ud | Cada base o salida trifasica |
| Toma TV / datos / telefono | ud | Cada toma terminal |
| Circuito dedicado | ud | Horno, termo, climatizacion, bomba, maquinaria, recarga EV |
| Cuadro electrico | ud | Cuadro general, secundario o de zona |
| Proteccion electrica | ud | PIA, diferencial, sobretensiones, magnetotermico, contactor |
| Canalizacion | ml | Tubo, bandeja o canaleta si se mide aparte |
| Cableado | ml | Por tipo y seccion cuando se mida aparte |

## Datos a extraer

- Numero de estancias y plantas.
- Puntos de luz por estancia.
- Mecanismos de mando por estancia.
- Tomas de corriente normales y especiales.
- Cuadros y subcuadros.
- Potencias o cargas singulares.
- Zonas humedas, exteriores o estancas.
- Tipo de instalacion: empotrada, vista, bandeja, canaleta, tubo rigido.

## Criterios especificos

- Un punto de luz no equivale necesariamente a una luminaria: una salida puede alimentar varias luminarias o una luminaria puede tener varios encendidos.
- Un marco con dos enchufes cuenta como dos tomas si son utilizables de forma independiente.
- Las tomas especiales se separan de los enchufes generales.
- Las luminarias de emergencia se cuentan aparte.
- Los cuadros se cuentan por envolvente y se desglosan sus protecciones si el alcance lo requiere.
- En exterior o zonas humedas, marcar material estanco y grado IP si aparece.

## Checklist

- [ ] Puntos de luz separados de luminarias.
- [ ] Mecanismos separados de enchufes.
- [ ] Tomas especiales identificadas.
- [ ] Cuadros y protecciones contados.
- [ ] Canalizacion y cableado no duplicados.
- [ ] Zonas humedas/exteriores marcadas.
- [ ] Certificado o boletin indicado como incluido o excluido.
