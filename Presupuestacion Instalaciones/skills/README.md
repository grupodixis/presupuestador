# Skills de Conteo

Las skills indican como debe razonar el agente cuando cuenta cada tipo de instalacion.

## Lista

| Archivo | Uso |
|---|---|
| `skill_conteo_general.md` | Metodo comun para cualquier medicion de instalaciones |
| `skill_electricidad.md` | Conteo de puntos electricos, cuadros, circuitos y canalizaciones |
| `skill_domotica.md` | Conteo de sensores, actuadores, pantallas, escenas y buses |
| `skill_iot.md` | Conteo de dispositivos conectados, gateways, comunicaciones y alimentacion |
| `skill_fontaneria.md` | Conteo de puntos de agua, ACS, saneamiento, llaves y equipos |

## Uso

1. Cargar primero `skill_conteo_general.md`.
2. Cargar despues la skill especifica de la disciplina.
3. Si hay varias disciplinas, medirlas por separado y cerrar con una tabla resumen.
