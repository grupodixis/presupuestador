# Skill: Conteo de Fontaneria

## Cuándo usar esta skill

Para contar instalaciones de agua fria, ACS, saneamiento, griferia, sanitarios, llaves, colectores, bombas, termos, riego basico y puntos hidraulicos.

## Unidades habituales

| Elemento | Unidad | Criterio de conteo |
|---|---:|---|
| Punto de agua fria | punto | Cada salida o consumo de AF |
| Punto de ACS | punto | Cada salida o consumo de agua caliente |
| Punto de desague | punto | Cada evacuacion conectada a saneamiento |
| Aparato sanitario | ud | Inodoro, lavabo, ducha, bañera, bidet, urinario |
| Griferia | ud | Cada grifo, mezclador, termostatica o valvula visible |
| Llave de corte | ud | General, estancia, aparato o circuito |
| Colector | ud | Cada colector de distribucion |
| Tuberia AF/ACS | ml | Por diametro y material si se mide aparte |
| Tuberia saneamiento | ml | Por diametro y material si se mide aparte |
| Bote sifonico / sifon | ud | Cada elemento sifonico |
| Termo / calentador / bomba | ud | Cada equipo productor o impulsor |
| Punto de riego | punto | Cada toma, electrovalvula o sector segun alcance |

## Datos a extraer

- Numero de baños, cocinas, lavaderos y zonas exteriores.
- Aparatos sanitarios por estancia.
- Puntos de AF, ACS y desague por aparato.
- Equipo de ACS: termo, caldera, aerotermia, solar.
- Material de tuberia si esta definido.
- Trazado empotrado, visto, falso techo, suelo o exterior.
- Diametros indicados en planos.
- Pendientes y registros en saneamiento.

## Criterios especificos

- Un lavabo normalmente implica AF, ACS y desague; contar las tres partidas si el alcance es completo.
- Un inodoro suele implicar AF y desague; no ACS salvo caso especial.
- Una ducha o bañera implica AF, ACS, desague, griferia y, si procede, mampara o valvula termostatica aparte.
- Separar tuberia de presion y saneamiento.
- Contar llaves generales y llaves por estancia/aparato cuando aparezcan o sean necesarias.
- En saneamiento, contar registros, botes sifonicos y arquetas si forman parte del alcance.

## Checklist

- [ ] AF, ACS y desague separados.
- [ ] Aparatos sanitarios contados.
- [ ] Griferia separada de sanitarios.
- [ ] Llaves, colectores y valvulas contados.
- [ ] Tuberias por material/diametro si se miden en ml.
- [ ] Saneamiento y registros considerados.
- [ ] Pruebas de presion y estanqueidad incluidas o excluidas.
