# Glosario de Conteo de Instalaciones

## AF

Agua fria sanitaria.

## ACS

Agua caliente sanitaria.

## Actuador

Elemento que ejecuta una accion: encender una luz, mover una persiana, abrir una valvula o activar un rele.

## Canal

Salida independiente de control. En domotica, un modulo fisico puede tener varios canales.

## Gateway

Equipo que comunica dispositivos o protocolos distintos, por ejemplo Zigbee a IP o KNX a IP.

## IoT

Internet of Things. Dispositivos conectados para medir, controlar o comunicar datos.

## Punto

Unidad funcional de instalacion: una toma, salida, consumo, sensor, desague o alimentacion terminal.

## Punto de luz

Salida electrica prevista para alimentar una luminaria o grupo de luminarias.

## Punto de agua

Salida hidraulica de agua fria o caliente para alimentar un aparato o consumo.

## Saneamiento

Red de evacuacion de aguas residuales o pluviales.

## Supuesto

Cantidad o criterio usado provisionalmente por falta de informacion completa. Debe quedar documentado.
