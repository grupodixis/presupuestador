# Plantillas

Plantillas de salida para los presupuestos generados por el agente IA.

## Formatos disponibles

| Archivo | Propósito |
|---|---|
| `presupuesto-simple.md` | Presupuesto básico para cliente particular |
| `presupuesto-desglosado.md` | Presupuesto detallado con todas las partidas |
| `presupuesto-licitacion.md` | Presupuesto para licitación pública |
| `presupuesto-tecnico.md` | Presupuesto con memoria técnica adjunta |

## Uso

El agente debe elegir la plantilla según el tipo de cliente y la complejidad del trabajo. Rellenar los campos con los datos calculados siguiendo el flujo de presupuestación.

Cada plantilla incluye marcadores de posición que el agente debe sustituir.
