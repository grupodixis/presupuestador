# Skill: Solicitud de Presupuestos a Proveedores

## Cuándo usar esta skill

Cuando faltan precios reales de materiales, tratamientos, transporte, alquileres o partidas subcontratadas para cerrar un presupuesto con costes defendibles.

Se usa antes de cerrar el coste técnico base y antes de aplicar márgenes finales.

## Datos mínimos necesarios

- Producto o partida que se está presupuestando.
- Material o servicio pendiente de precio.
- Cantidad estimada y unidad.
- Especificación técnica mínima.
- Formato deseado de respuesta: precio unitario, plazo, transporte, IVA incluido o no incluido.

## Datos recomendados

- Referencia del proyecto u obra.
- Plano, croquis o medidas relevantes.
- Calidad, normativa o certificado requerido.
- Lugar de entrega.
- Fecha límite para recibir oferta.
- Proveedor habitual o tipo de proveedor a contactar.

## Criterios

- No pedir solo precio: pedir también plazo, disponibilidad, transporte, condiciones de pago e IVA.
- Separar partidas si el proveedor puede cotizarlas por líneas independientes.
- Indicar claramente si la cantidad es estimada o cerrada.
- Solicitar alternativas si pueden mejorar precio o plazo sin bajar calidad.
- Pedir confirmación expresa de que la oferta cumple la especificación.

## Materiales y servicios típicos a solicitar

- Acero inoxidable, acero carbono, aluminio o perfiles especiales.
- Vidrio, policarbonato, panel sandwich, brezo, toldos y cubiertas.
- Pintura, lacado, galvanizado, chorreado o anodizado.
- Tornillería, anclajes, tacos químicos y herrajes.
- Transporte, grúa, plataforma elevadora o andamio.
- Subcontratas de montaje o tratamientos externos.

## Proceso de trabajo

1. Detectar qué costes no tienen precio fiable en las tablas internas.
2. Convertir la necesidad técnica en una lista clara de partidas.
3. Elegir canal: email para solicitudes formales, WhatsApp para consultas rápidas.
4. Redactar la solicitud usando la plantilla correspondiente.
5. Incluir datos mínimos para que el proveedor no tenga que repreguntar.
6. Pedir respuesta desglosada y plazo de validez.
7. Al recibir respuesta, registrar precio, fecha, proveedor y condiciones.
8. Actualizar el presupuesto con el coste real o marcarlo como pendiente.

## Formato de salida recomendado

El agente debe entregar uno o varios mensajes listos para enviar:

1. **Email formal** si se requiere oferta documentada.
2. **WhatsApp breve** si se busca precio rápido o disponibilidad.
3. **Resumen interno** con partidas solicitadas y datos pendientes.

## Checklist final

- [ ] Proveedor identificado o tipo de proveedor indicado.
- [ ] Partidas cuantificadas.
- [ ] Especificación técnica clara.
- [ ] Lugar de entrega indicado.
- [ ] Plazo de respuesta solicitado.
- [ ] Se pide transporte por separado si aplica.
- [ ] Se pide plazo de entrega.
- [ ] Se pide IVA incluido/excluido.
- [ ] Se pide validez de la oferta.
- [ ] Mensaje listo para enviar sin texto interno del agente.

## Errores frecuentes a evitar

- Pedir precio sin cantidades.
- No indicar calidad o material exacto.
- No pedir plazo de entrega.
- No aclarar si el transporte está incluido.
- Mezclar dudas internas con texto para el proveedor.
- Enviar al cliente una solicitud pensada para proveedor.
