# Skill: Oferta y Demanda (Ajuste de Precio)

## Cuándo usar esta skill

Después de calcular el coste técnico base y aplicar el margen industrial. Se usa para ajustar el precio final según la capacidad productiva actual y las condiciones del mercado. Es la penúltima skill antes de la revisión final.

## Datos mínimos necesarios

- Carga actual del taller (porcentaje estimado).
- Volumen de trabajo pendiente (horas o semanas).
- Plazo de entrega solicitado por el cliente.
- Urgencia del cliente.
- Tipo de cliente (particular, empresa, administración, habitual, nuevo).

## Datos recomendados

- Historial de precios de la competencia (estimación).
- Época del año (temporada alta/baja).
- Relación de trabajos similares previos con precio final.
- Presupuesto del cliente (si lo ha indicado).
- Número de ofertantes en licitación (si se conoce).
- Margen de negociación disponible.

## Criterios técnicos

- El ajuste por oferta/demanda se aplica sobre el precio con margen industrial, no sobre el coste.
- El ajuste debe ser razonable y justificable ante el cliente.
- El precio final debe tener un rango de negociación implícito.
- No se debe bajar el precio por debajo del coste técnico base.
- En temporada baja se puede reducir margen para mantener carga.
- En temporada alta se puede aumentar margen para maximizar rentabilidad.
- El cliente habitual puede tener condiciones especiales.

## Factores de ajuste

| Factor | Ajuste sobre precio con margen |
|---|---|
| Taller < 60% carga | -5% a -10% |
| Taller 60-80% carga | 0% (estándar) |
| Taller 80-100% carga | +5% a +10% |
| Taller saturado >100% | +15% a +25% |
| Urgencia alta | +10% a +20% |
| Sin urgencia | 0% |
| Cliente habitual | -5% |
| Cliente nuevo | 0% |
| Cliente estratégico | -5% a -10% |
| Cliente administración | 0% (precio cerrado) |
| Temporada baja | -5% a -10% |
| Temporada alta | +5% a +10% |
| Gran volumen | -5% a -15% |
| Pequeño trabajo | 0% a +5% |

## Proceso de ajuste

1. Obtener el precio técnico final (coste base + margen).
2. Determinar carga actual del taller.
3. Determinar urgencia del cliente.
4. Determinar tipo de cliente.
5. Determinar temporada.
6. Evaluar volumen del trabajo.
7. Aplicar ajustes acumulativos (sumar/restar porcentajes).
8. Verificar que el precio final es razonable.
9. Dejar margen de negociación (5-10% sobre el precio presentado).

## Estrategias según contexto

### Capacidad baja (carga < 60%)
- Priorizar captación de trabajo.
- Ofrecer descuento por contratación rápida.
- No aplicar recargos por urgencia.
- Flexibilizar condiciones de pago.

### Capacidad alta (carga > 80%)
- Priorizar trabajos con mayor margen.
- Aplicar recargo por urgencia.
- Ser selectivo con trabajos pequeños o complejos.
- Mantener condiciones de pago estándar.

### Cliente habitual
- Ofrecer precio preferente.
- Mantener condiciones de pago flexibles.
- Priorizar su plazo de entrega.
- Considerar el margen anual del cliente.

### Licitación pública
- Precio cerrado y competitivo.
- No aplicar recargos.
- Incluir todos los costes indirectos.
- Revisar pliegos rigurosamente.

## Errores frecuentes a evitar

- Aplicar descuentos sin verificar la carga de trabajo.
- Bajar precio por debajo del coste sin justificación comercial.
- No dejar margen de negociación.
- Aplicar recargos excesivos que ahuyenten al cliente.
- No considerar el valor a largo plazo del cliente.
- Ignorar la competencia en licitaciones.

## Preguntas que el agente debe hacerse

- ¿Qué carga de trabajo tenemos ahora?
- ¿Cuándo necesita el cliente el trabajo?
- ¿Es un cliente habitual?
- ¿Hay competencia en este trabajo?
- ¿Es temporada alta o baja?
- ¿Tiene el cliente presupuesto límite?
- ¿Qué margen de negociación dejo?
- ¿Este trabajo puede abrir puertas a otros clientes?

## Checklist final

- [ ] Carga de trabajo actual determinada.
- [ ] Urgencia del cliente evaluada.
- [ ] Tipo de cliente considerado.
- [ ] Temporada evaluada.
- [ ] Ajuste aplicado justificado.
- [ ] Margen de negociación reservado.
- [ ] Precio final razonable y competitivo.
- [ ] Condiciones de pago definidas.

## Formato de salida recomendado

El ajuste de oferta/demanda debe documentarse en el presupuesto como nota o como partida justificativa, sin desglosar al cliente el factor exacto. Se puede indicar como:

- "Descuento especial por cliente habitual"
- "Recargo por urgencia en la ejecución"
- "Precio preferente por volumen"
- O incluirse directamente en el precio final sin anotación.
