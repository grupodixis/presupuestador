# Ejemplos

Ejemplos completos de presupuestos generados por el agente IA.

## Propósito

Servir como referencia y casos de prueba para validar que el agente sigue correctamente el flujo de presupuestación.

## Estructura de cada ejemplo

Cada ejemplo debe incluir:

- Descripción del trabajo solicitado por el cliente.
- Razonamiento paso a paso del agente.
- Skill(s) utilizadas.
- Composición(es) consultadas.
- Cálculos de costes.
- Evaluación de complejidad y riesgos.
- Margen aplicado y justificación.
- Ajuste de oferta/demanda y justificación.
- Presupuesto final generado.
- Notas y dudas planteadas al cliente.

## Ejemplos a desarrollar

- `ejemplo-barandilla-inox.md` — Barandilla de acero inoxidable, ambiente marino.
- `ejemplo-estructura-nave.md` — Estructura metálica para nave industrial.
- `ejemplo-puerta-cortafuego.md` — Puerta metálica cortafuego.
- `ejemplo-cerramiento-aluminio.md` — Cerramiento de aluminio con vidrio.
- `ejemplo-instalacion-electrica.md` — Instalación eléctrica completa.
- `ejemplo-escalera-caracol.md` — Escalera metálica en caracol a medida.
