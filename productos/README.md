# Productos

Catálogo técnico de los productos que la empresa fabrica e instala.

## Subcarpetas

| Carpeta | Propósito |
|---|---|
| `composiciones/` | Composición típica de cada producto (materiales, procesos, costes) |
| `reglas-tecnicas/` | Reglas técnicas, tolerancias, normas aplicables por tipo de producto |

## Uso

El agente debe consultar la composición del producto identificado para conocer:

- Materiales principales y auxiliares.
- Procesos de fabricación involucrados.
- Variables técnicas que afectan al precio.
- Factores de complejidad.
- Costes a considerar.

Las reglas técnicas contienen conocimiento normativo y de diseño que el agente debe tener en cuenta al validar la viabilidad del presupuesto.
