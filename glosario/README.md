# Glosario

Definiciones de términos técnicos y comerciales utilizados en el entorno de presupuestación.

## Propósito

- Asegurar que el agente IA utiliza la terminología correcta.
- Evitar ambigüedades en la comunicación con el cliente.
- Unificar criterios entre diferentes skills y composiciones.
- Facilitar la incorporación de nuevos productos al catálogo.

## Estructura de cada entrada

```markdown
### Término

Definición clara y concisa.

**Contexto de uso:** ¿Cuándo aparece este término?

**Relacionados:** lista de términos relacionados.
```

## Contenido sugerido

Se deben documentar al menos estos términos:

- **Presupuesto cerrado / presupuesto estimado / presupuesto abierto**
- **Coste técnico base**
- **Margen industrial / margen comercial**
- **Factor de complejidad**
- **Factor de riesgo**
- **Merma**
- **Replanteo**
- **Medición**
- **Ambiente C1-C5** (según ISO 12944)
- **Tratamiento superficial / imprimación / pintura / galvanizado / anodizado**
- **Soldadura MIG/MAG / TIG / electrodo**
- **Carga de trabajo / capacidad productiva**
- **Plazo de entrega / urgencia**
- **Partida / unidad de obra**
- **Contrato de obra / contrato mercantil**
- **Certificado de garantía / DNI / marcado CE**
